# 💸 SmartCash - Mobile Financial Tracker

A comprehensive mobile financial application designed to simplify personal finance management. This app features intuitive expense analysis, helping users to easily recognize their cash flow patterns, monitor daily financial activities, and make better data-driven financial decisions.

> **Note:** This repository is a forked archive of a collaborative team project.

## 🚀 Tech Stack
* **Frontend (Mobile):** Flutter / Dart
* **Backend:** Node.js
* **Database:** PostgreSQL
* **State Management:** Provider
* **Architecture:** MVC (Model-View-Controller) Pattern

## 💡 Key Features
1. **Dashboard & Cashflow Overview:** A visual summary of total income, expenses, and remaining balance for the current month.
2. **Smart Expense Analysis:** Categorizes transactions automatically (e.g., Food, Transport, Bills) and visualizes them using interactive charts.
3. **Transaction Management:** Seamlessly add, edit, or delete daily financial activities.
4. **Historical Reports:** Filter and view past financial data to track long-term financial health.
